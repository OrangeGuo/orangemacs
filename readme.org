* OrangEmacs
a pure and quick configuration for emacs
** install
1. git clone this repository
2. start emacs 
  
** dependency
- emacs
- git for magit 
- mlocate for counsel-locate
- silversearcher-ag for helm-ag
- font:source code pro,nsimsun
